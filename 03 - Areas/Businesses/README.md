## Separation from Clients

**Clients** (`02 - Projects/Active Clients/`) = people who pay us for Ecosire's capabilities.
**Businesses** (this folder) = the capabilities themselves as product lines.

A single client may consume multiple businesses (e.g., Amalfi = Managed Hosting + custom Odoo Modules + ECOSIRE.COM consulting hours).

## Rule for new entries

When a new revenue stream emerges (e.g., a new SaaS, a new services line), create a folder here with a README covering:

1. **Purpose** — 1-2 sentence product statement
2. **Where the code lives** — sibling workspace path
3. **Pricing** — current tiers
4. **Positioning** — who is this for, what alternatives do they have
5. **Current focus** — short-term roadmap / active initiatives
6. **Cross-workspace references** — pointer to sibling CLAUDE.md
